# 💱Currency Converter
A simple and responsive Currency Converter web application built using HTML, CSS, and JavaScript. This project allows users to convert one currency into another using real-time exchange rates.

# 🚀Features
- Convert between multiple currencies
- Built with HTML, CSS, and JavaScript
- Real-time exchange rate support

# 🛠️Technologies Used
- **HTML5** - For structure of the web
- **CSS3** - For styling and layout
- **JavaScript(ES6)** - For logic and Interactivity
- **Currency Exchange API** - For Exchange rates
